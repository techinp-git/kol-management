# Documentation

เอกสารและคู่มือของโปรเจกต์ KOL Management

## โฟลเดอร์

- **ราก `docs/`** – คู่มือตั้งค่า, แก้ปัญหา, สรุปโมดูล (ไฟล์ `.md`)
- **sql/** – สคริปต์ SQL ชั่วคราว (quick fix, check data)
- **examples/** – ไฟล์ตัวอย่าง / ทดสอบ (เช่น HTML สำหรับทดสอบ pagination)

## เอกสารหลัก (ตัวอย่าง)

- `SUPABASE_CONNECTION.md` – การเชื่อมต่อ Supabase
- `AUTH_USERS_GUIDE.md` – การใช้งาน Auth
- `TROUBLESHOOTING.md` – แก้ปัญหาเบื้องต้น
- `SETUP_*.md` – คู่มือตั้งค่าต่างๆ (KOL, Campaigns, Account Channels ฯลฯ)
- `KOL_*.md` – เรื่อง KOL module (checklist, summary, search, pagination ฯลฯ)
